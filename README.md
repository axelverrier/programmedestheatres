Ce site donne une vision globale et chronologique des pièces actuellement jouées à Paris et aux alentours en Ile-de-France.
Il n'a pas (encore) vocation à être exhaustif.

Ce projet est librement inspiré de www.programmedesexpos.paris de Terry Laire.
Il est entièrement libre de droit.

Si vous constatez des erreurs ou des manquements dans les données ou que vous avez des suggestions, vous pouvez créer une issue sur Github.
