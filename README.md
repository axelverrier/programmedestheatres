Ce site vise à donner une vision globale et chronologique des pièces jouées à Paris et aux alentours en Ile-de-France. 
Il n'a pas (encore) vocation à être exhaustif.

Ce projet est librement inspiré de www.programmedesexpos.paris de Terry Laire. 
Il est entièrement libre de droit puisqu'il est autant mon fait que celui de Claude Code, si ce n'est davantage le sien. 

Si vous constatez des erreurs ou des manquements dans les données ou que vous avez des suggestions, vous pouvez:
- me contacter, 
- créer une issue sur Github, ou mieux: une pull request.
